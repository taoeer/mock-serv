# mock-serv
A simple nodejs based mock data server width reset full api for development!

# Installation
``npm install mock-serv``
or
``yarn add mock-serv``

# Command
 ``ms`` or ``mock-server``

# Optins
-v version

-c Specified the config file, default is config.js (a nodejs module)

-p Specified the server port

-h Show help
